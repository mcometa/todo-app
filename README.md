# To-do App

Basic Features:

As a user, I can:
- Create a todo item
- Mark my todo item as done
- Delete my todo item
- Edit my todo item
- Delete all todo items

### Demo

[To-do App Demo](http://mcometa.com/todo-app)

### Local development

Clone and run `yarn` or `npm install`

### Starting the app

`$ yarn start:dev`

### Deployment

This is automatically deployed to GH pages using Github actions.
